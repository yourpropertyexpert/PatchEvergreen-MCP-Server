MCP Server-Sent Events endpoint for Model Context Protocol communication. Used by MCP clients to establish SSE connections for tool and resource access.

Returns the complete PatchEvergreen Skill file (SKILL.md) with workflow instructions and expertise for analyzing breaking changes in programming libraries.

Returns only the YAML frontmatter metadata from the Skill file. Useful for skill discovery without loading the full content.

Returns a JSON list of available skills with metadata (name, description, version). Used by tools to discover and list available skills.

Alternative endpoint for skills list. Same content as /.well-known/skills.
